Important notice
  • All entities, events, metrics, and facts are fictional.
  • This repository exists solely for AI retrieval and grounding evaluation.
  • No content should be interpreted as real-world information.

About this repository

This repository hosts a controlled fictional knowledge base used to test document ingestion, chunking strategies, retrieval accuracy, and hallucination resistance in RAG pipelines.

Each language version exposes equivalent content at a stable URL, enabling consistent multilingual indexing and retrieval experiments.

Intended RAG use cases

  • Semantic and keyword retrieval comparison
  • Multi-hop reasoning across sections
  • Grounded question answering
  • Hallucination detection when answers are absent
  • Cross-language retrieval consistency
